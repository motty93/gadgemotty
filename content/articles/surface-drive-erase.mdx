古いMicrosoft Surface Pro 5を処分しようと思い、その前にNVME SSDのデータを削除しなければならないことに思い当たりました。wipe-outなどのソフトウェアによる完全消去がありますが、Surfaceには専用のMicrosoft Surface Data Eraserというツールがあります。

## Microsoft Surface Data Eraserとは

Microsoft Surface Data Eraserは、Microsoft社が提供する専用のデータ消去ツールです。Surface製品のSSDやHDDを完全に消去するための公式ツールで、USBドライブから起動して使用します。このツールを使用することで、Surface上のすべてのデータを完全に消去し、復元不可能にすることができます。

## 準備するもの

- USB 3.0フラッシュドライブ（8GB以上）
- 別のWindows PCでの作業
- Surface Data Eraserツール（Microsoft公式サイトからダウンロード）
- 処分したいSurface製品と電源アダプター

## Surface Data Eraserの使い方

1. **準備：** 公式サイトからSurface Data Eraserツールをダウンロードします。
2. **USBの作成：** 別のWindowsコンピューターで、Surface Data Eraserツールを実行し、USBブートメディアを作成します。
3. **起動設定：** Surfaceの電源を切り、USBを接続します。
4. **起動：** 電源ボタンとボリュームアップボタンを同時に押しながらSurfaceの電源を入れ、UEFI画面を表示します。
5. **ブート順序の変更：** UEFIでUSBからの起動を選択します。
6. **データ消去：** Surface Data Eraserが起動したら、画面の指示に従ってデータ消去を実行します。

## 注意点

- データ消去プロセスは元に戻せないため、重要なデータはあらかじめバックアップしてください。
- Surface Data Eraserは、すべてのSurfaceモデルで動作するとは限りません。対応モデルを確認してください。
- プロセスには数時間かかる場合があります。その間、電源アダプターを接続しておくことをお勧めします。
- Windows BitLockerが有効になっている場合は、事前に無効にするか、回復キーを準備しておく必要があります。

## 代替方法

Surface Data Eraserが使用できない場合や、より高度なデータ消去が必要な場合は、以下の代替方法があります：

1. **Windowsのリセットとドライブのクリーンアップ：** Windowsの設定から「PCをリセット」を選択し、「ファイルの削除とドライブのクリーンアップ」オプションを使用する。
2. **サードパーティのデータ消去ツール：** DBANやShredOSなどのサードパーティツールを使用する。

## まとめ

Microsoft Surface Data Eraserは、Surface製品を処分する前に、個人データをセキュアに消去するための簡単かつ効果的な方法です。重要なデータがSurfaceに保存されている場合は、処分前に必ずこのようなツールを使用してデータを消去することをお勧めします。
